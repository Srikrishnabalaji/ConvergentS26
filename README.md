# WavePoint (Convergent BT Spring 2026)

A mobile app that helps you easily navigate inside campus buildings. Search for a building, pick the room you need, and get an indoor path generated by our CV model overlaid on the floor plan — gone are the days of endlessly wandering hallways. Also includes friend groups, campus org discovery, a shared calendar, and location sharing.

## Tech Stack

- **Frontend:** React Native, Expo, TypeScript, NativeWind (Tailwind CSS)
- **Navigation:** Expo Router (file-based), React Navigation
- **Backend:** Supabase (PostgreSQL, Auth, Storage, RLS)
- **Maps:** React Native Maps, OpenStreetMap geocoding
- **CV / Pathfinding:** Python (floorplan parsing, grid-based navigation)
- **Notifications:** Expo Notifications

## Setup

You'll need Node.js (v18+), npm, and either Expo Go on your phone or an iOS/Android simulator.

```bash
cd frontend
npm install
```

## Running the App

```bash
npx expo start --tunnel
```

We use `--tunnel` so it works across different networks without firewall headaches. Once it's running:

- Scan the QR code with Expo Go on your phone
- Or press `i` for iOS simulator / `a` for Android emulator

To run natively (requires Xcode or Android Studio):

```bash
npm run ios
npm run android
```

## Project Structure

```
frontend/       React Native app (Expo Router, NativeWind)
backend/        Python scripts for floorplan parsing
supabase/       Database schema and migrations
```

## Environment Variables

Create a `frontend/.env` file with your Supabase credentials:

```
EXPO_PUBLIC_SUPABASE_URL=your_url_here
EXPO_PUBLIC_SUPABASE_ANON_KEY=your_key_here
```
